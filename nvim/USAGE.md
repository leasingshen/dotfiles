# Neovim 配置使用说明

> Leader 键为 `Space`

---

## 文件操作

| 快捷键 | 功能 |
|--------|------|
| `<Space>w` | 保存文件 |
| `<Space>q` | 退出 |
| `<Esc>` | 清除搜索高亮 |

---

## 窗口与 Buffer

**窗口跳转**（`Ctrl` + 方向）：

| 快捷键 | 功能 |
|--------|------|
| `Ctrl-h` | 跳到左侧窗口 |
| `Ctrl-j` | 跳到下方窗口 |
| `Ctrl-k` | 跳到上方窗口 |
| `Ctrl-l` | 跳到右侧窗口 |

**Buffer 切换**：

| 快捷键 | 功能 |
|--------|------|
| `Shift-l` | 下一个 buffer |
| `Shift-h` | 上一个 buffer |

---

## 编辑增强

| 快捷键 | 模式 | 功能 |
|--------|------|------|
| `Ctrl-d` | Normal | 向下翻半页（光标保持居中） |
| `Ctrl-u` | Normal | 向上翻半页（光标保持居中） |
| `J` | Visual | 将选中行下移 |
| `K` | Visual | 将选中行上移 |

### Surround（nvim-surround）

在文本对象周围增删改包围符号：

| 操作 | 示例效果 |
|------|----------|
| `ysiw"` | 给单词加双引号：`word` → `"word"` |
| `yss)` | 给整行加括号 |
| `ds"` | 删除最近的双引号：`"word"` → `word` |
| `cs'"` | 将单引号改为双引号：`'word'` → `"word"` |
| `S"` | Visual 模式下，给选中内容加双引号 |

---

## 文件树（nvim-tree）

| 快捷键 | 功能 |
|--------|------|
| `<Space>e` | 切换文件树显示/隐藏 |
| `<Space>o` | 聚焦到文件树 |

文件树内常用操作：

| 按键 | 功能 |
|------|------|
| `Enter` / `o` | 打开文件 |
| `a` | 新建文件/目录（以 `/` 结尾为目录） |
| `d` | 删除 |
| `r` | 重命名 |
| `x` | 剪切 |
| `c` | 复制 |
| `p` | 粘贴 |
| `H` | 显示/隐藏隐藏文件 |
| `q` | 关闭文件树 |

---

## 模糊搜索（Telescope）

| 快捷键 | 功能 |
|--------|------|
| `<Space>ff` | 搜索文件名 |
| `<Space>fg` | 全局内容搜索（live grep） |
| `<Space>fb` | 搜索已打开的 buffer |
| `<Space>fh` | 搜索帮助文档 |

Telescope 弹窗内：

| 按键 | 功能 |
|------|------|
| `Ctrl-n` / `Ctrl-p` | 上下移动结果 |
| `Enter` | 打开选中文件 |
| `Ctrl-v` | 垂直分割打开 |
| `Ctrl-x` | 水平分割打开 |
| `Esc` | 关闭 |

---

## 代码跳转（LSP）

| 快捷键 | 功能 |
|--------|------|
| `gd` | 跳转到定义 |
| `gD` | 跳转到声明 |
| `gi` | 跳转到实现 |
| `gr` | 查看所有引用 |
| `K` | 查看悬浮文档 |
| `<Space>lr` | 重命名符号 |
| `<Space>ca` | 代码 Action（快速修复等） |
| `<Space>lf` | 格式化当前文件 |

**诊断跳转**：

| 快捷键 | 功能 |
|--------|------|
| `[d` | 上一个诊断错误 |
| `]d` | 下一个诊断错误 |
| `<Space>ld` | 查看当前行诊断详情 |
| `<Space>lx` | 将诊断列表写入 location list |

---

## 语法文本对象（Treesitter Textobjects）

在 Normal / Visual / Operator 模式下，基于语法树选取代码块：

| 按键 | 选取范围 |
|------|----------|
| `af` | 整个函数（含签名） |
| `if` | 函数体（不含签名） |
| `ac` | 整个类 |
| `ic` | 类体 |
| `aa` | 整个参数（含逗号） |
| `ia` | 参数值本身 |

示例：`daf` 删除整个函数，`vif` 选中函数体，`caa` 修改一个参数。

**函数/类跳转**：

| 按键 | 功能 |
|------|------|
| `]f` | 下一个函数开头 |
| `[f` | 上一个函数开头 |
| `]c` | 下一个类开头 |
| `[c` | 上一个类开头 |

---

## Git（Gitsigns）

| 快捷键 | 功能 |
|--------|------|
| `]h` | 下一个修改块（hunk） |
| `[h` | 上一个修改块 |
| `<Space>gp` | 预览当前 hunk |
| `<Space>gs` | 暂存当前 hunk |
| `<Space>gu` | 撤销暂存当前 hunk |
| `<Space>gr` | 重置当前 hunk（丢弃修改） |
| `<Space>gb` | 显示当前行的 git blame |

---

## 格式化

保存文件时自动格式化（由 conform.nvim 驱动）。

| 快捷键 | 功能 |
|--------|------|
| `<Space>tf` | 切换自动格式化开/关 |
| `<Space>lf` | 手动格式化当前文件 |

各语言格式化工具：

| 语言 | 工具 |
|------|------|
| Lua | stylua |
| Python | isort + black |
| JS / TS | prettier |
| JSON / Markdown | prettier |
| Shell | shfmt |

---

## 补全（nvim-cmp）

| 按键 | 功能 |
|------|------|
| `Ctrl-Space` | 手动触发补全 |
| `Tab` | 选择下一项 / 展开或跳转 snippet |
| `Shift-Tab` | 选择上一项 / 反向跳转 snippet |
| `Enter` | 确认选中项（需主动选中才生效） |
| `Ctrl-e` | 关闭补全菜单 |

---

## 工具面板

| 快捷键 | 功能 |
|--------|------|
| `<Space>tt` | 打开 Lazy（插件管理） |
| `<Space>tm` | 打开 Mason（LSP/工具安装） |

---

## Which-key

任意 Normal 模式下，按下 `<Space>` 后稍等片刻，会弹出按键提示面板，
显示所有以 Leader 开头的可用快捷键及其描述。
